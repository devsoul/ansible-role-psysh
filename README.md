Psysh Ansible role
=========

If you are a PHP developer and you use Psysh.

Requirements
------------
  - `php >= 5.4` 

Role Variables
--------------
None

Dependencies
------------
None

Example Playbook
----------------
    - hosts: php
      roles:
        - devsoul.psysh

Check if it's installed:
------
    psysh --version
